# FlashAir Password

Default FlashAir password and ways to modify it.

# What's the default password of FlashAir Wi-Fi?

The default password is: 12345678.

# How can I change the password for FlashAir password?

There are 4 ways to modify the FlashAir password.

  1. Use the Official FlashAir Configure Tool

Open the software, you can change the SSID and password in the Settings. Notice, the password should be at least 8 length. If you delete the default password and also don't set a new one, it means that your FlashAir Wi-Fi can be connected by any deevice.
You can download the software below (it comes from the Official site):

Windows Version

Mac Version

  1. Use Browser

When your FlashAir has not been connected, you can use this way to modify the password.
After buying the FlashAir, your device, like PC or Mobile, connects to the FlashAir successfully, and then in the browser you input:

http://flashair (opens new window)

then you can follow the instructions.

  1. Use FlashAir App

You can use the Official app or any 3-rd party app. Notice: it needs the master code for certain 3rd-party app for modifying password.

  1. Modify CONFIG directly

Input the FlashAir into the slot, and find the CONFIG file of the FlashAir disk. You can open it using any text supported software. Find APPNETWORKKEY and replace the *** after the =. It also requires at least 8 length. Save the file and close it, and then inject the FlashAir disk.

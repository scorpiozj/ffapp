本教程包含如何使用FlashAir SD 卡，以及如何使用FlashAir App访问卡片内容。

# 1. 启用FlashAir无线热点

只需将卡插入相机，打开电源。对的，就这么简单！

# 2. 连接到FlashAir无线热点

以智能手机连接为例，打开Wi-Fi设置，在Wi-Fi列表中找到以“flashair”为前缀的无线网，点击连接。注意：初始密码为“12345678”。

# 3. 访问FlashAir卡片内容

当你完成了第二步以后，你的智能手机就可以通过以下的方式读取卡片内容。

# 3.1 通过浏览器

打开智能手机的浏览器(比如，iPhone上的Safari浏览器)，然后输入如下网址：

http://flashair (opens new window)

然后会出现如下的页面：

接下来你就可以根据页面进行操作。

# 3.2 通过官方的FlashAir app

现如今是一个智能时代，KIOXIA 公司开发了名为 FlashAir(iOS Version (opens new window), Android Version (opens new window))的App，它提供了访问FlashAir的功能。下图是App的界面(以iOS为例)：

左边是设备未连接到FlashAir无线热点时app内的界面，在这个界面上提供了连接到FlashAir热点的方法。这里需要注意的是，对iOS用户，该app需要用户授予它访问位置的权限。
当设备连接到FlashAir热点后且赋予app访问位置的权限后，app的界面如右图所示。这是一个常规的文件视图界面，你可以点击文件夹访问其文件列表，也可以点击图片————这时它会弹出一个新页面，显示下载的进度。当下载完成后，就可以看到图片。大多数图片格式，如jpg/png/heic，app都能支持。常见的RAW格式，app也能打开。

# 3.3 by 3-rd party App: FlashAir Share

对于一款工具型app需要位置权限，有些人会觉得不可思议。幸好有很多第三方的app可以供大家使用，其中之一是 FlashAir Share (opens new window)。

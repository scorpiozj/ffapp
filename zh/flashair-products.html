# FlashAir SD 产品

# 可以使用 FlashAir SD 卡的设备

FlashAir SD 卡包含了 flash 闪存，它在完全遵循SD卡的各种标准情况下，添加了支持无线Lan的芯片和天线。因此可以视作一个标准的SD内存卡使用在电子设备上，比如带有 SDHC™ 槽的笔记本电脑等。

# FlashAir 替代品

非常非常遗憾：并没有所谓的 FlashAir SD 卡的替代品。 一方面因为东芝财务作假经营不善，另一方面许多相机已经内置 Wi-Fi，因此类卡的市场越变越小。

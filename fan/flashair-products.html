# FlashAir SD 產品

# 可以使用 FlashAir SD 卡的設備

FlashAir SD 卡包含了 flash 閃存，它在完全遵循SD卡的各種標準情況下，添加了支持無線Lan的芯片和天線。因此可以視作一個標準的SD內存卡使用在電子設備上，比如帶有 SDHC™ 槽的筆記本電腦等。

# FlashAir 替代品

非常非常遺憾：並沒有所謂的 FlashAir SD 卡的替代品。一方面因為東芝財務作假經營不善，另一方面許多相機已經內置 Wi-Fi，因此類卡的市場越變越小。

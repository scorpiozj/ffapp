# FlashAir之CONFIG文件

CONFIG 文件包含了FlashAir卡片的核心配置。除非你清楚地了解這些配置，否則不要輕易修改該文件。

# CONFIG簡介

CONFIG文件上存儲了 FlashAir 的相關配置，比如FlashAir AP模式點的無線接入點名稱等等。它位於文件夾 /SD_WLAN 內，並且該文件的文件名不帶後綴，就是 CONFIG。並且該文件是一個隱藏文件，因此當你要在PC上查看或修改時，需要設置PC可以查看隱藏文件夾和文件。 注意：

  1. 記住一定要先複製文件內容再修改
  2. 記住一定要在PC上彈出FlashAir再將卡拔出

# FlashAir 使用指南

本教程包含如何使用FlashAir SD 卡，以及如何使用FlashAir App訪問卡片內容。

# 1. 啟用FlashAir無線熱點

只需將卡插入相機，打開電源。對的，就這麼簡單！

# 2. 連接到FlashAir無線熱點

以智能手機連接為例，打開Wi-Fi設置，在Wi-Fi列表中找到以“flashair”為前綴的無線網，點擊連接。注意：初始密碼為“12345678”。

# 3. 訪問FlashAir卡片內容

當你完成了第二步以後，你的智能手機就可以通過以下的方式讀取卡片內容。

# 3.1 通過瀏覽器

打開智能手機的瀏覽器(比如，iPhone上的Safari瀏覽器)，然後輸入如下網址：

http://flashair (opens new window)

然後會出現如下的頁面：

接下來你就可以根據頁面進行操作。

# 3.2 通過官方的FlashAir app

現如今是一個智能時代，KIOXIA 公司開發了名為FlashAir(iOS Version (opens new window), Android Version (opens new window))的App，它提供了訪問FlashAir的功能。下圖是App的界面(以iOS為例)：

左邊是設備未連接到FlashAir無線熱點時app內的界面，在這個界面上提供了連接到FlashAir熱點的方法。這裡需要注意的是，對iOS用戶，該app需要用戶授予它訪問位置的權限。 當設備連接到FlashAir熱點後且賦予app訪問位置的權限後，app的界面如右圖所示。這是一個常規的文件視圖界面，你可以點擊文件夾訪問其文件列表，也可以點擊圖片————這時它會彈出一個新頁面，顯示下載的進度。當下載完成後，就可以看到圖片。大多數圖片格式，如jpg/png/heic，app都能支持。常見的RAW格式，app也能打開。

# 3.3 by 3-rd party App: FlashAir Share

對於一款工具型app需要位置權限，有些人會覺得不可思議。幸好有很多第三方的app可以供大家使用，其中之一是 FlashAir Share (opens new window)。

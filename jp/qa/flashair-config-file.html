# FlashAir CONFIG ファイル

CONFIGファイルには、FlashAirのコア構成が含まれています。完全に理解していない限り、変更しないでください。

# CONFIG イントロ

これにはFlashAirの構成が含まれており、SDカードの/ SD_WLANフォルダーにあり、ファイル名は拡張子のないCONFIGです（つまり、単にCONFIGであり、CONFIG.txtではありません）。これは通常隠しファイルであるため、コンピューターでこれを表示および編集するには、隠しファイルとフォルダーがSDスロットに表示されることを許可していることを確認する必要があります。 慣れていない場合は、このファイルを変更しないでください。

知らせ：

  1. ファイルを編集する前に必ずコピーを作成してください
  2. FlashAirを常にコンピューターから取り出してから、スロットから取り外します

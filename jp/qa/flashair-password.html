# FlashAir パスワード

デフォルトのFlashAirパスワードとその変更方法。

# FlashAir Wi-Fiのデフォルトパスワードは何ですか？?

デフォルトのパスワードは12345678です。

# How can I change the password for FlashAirのパスワードを変更するには？

ワイヤレスパスワードを変更するには、次の4つの方法があります。

  1. 公式のFlashAir構成ツールソフトウェアを使用します

ソフトウェアを開き、「カード設定」メニューでワイヤレスアクセスポイントのパスワードと名前を変更できます。パスワードには少なくとも8文字が含まれている必要があることに注意してください。デフォルトのパスワードを削除し、パスワードを設定しない場合、FlashAirはパスワードを必要とせず、接続されているすべてのクライアントがカードにアクセスできることを意味します。 構成ツールソフトウェアのダウンロード（公式Webサイトからダウンロードしてください）：

Windowsバージョン

Macバージョン

  1. ブラウザの変更（最初の接続）

FlashAirを購入した後、初めてAPモードでFlashAirに接続し、ブラウザに入力します。

[http://flashair]（http://flashair）

新しく開いたページで、メニューを選択します。

  1. アプリの変更

公式またはサードパーティのアプリを使用して変更できます。一部のサードパーティアプリは、変更する前にFlashAirのマスターコードを取得する必要があることに注意してください。

4.CONFIGファイルを変更します

FlashAirをコンピューターに挿入し、FlashAirのCONFIGファイルを開いて、「APPNETWORKKEY」という行を見つけます。等号の後の「***」文字は現在のパスワードを示します。等号の後の内容を新しいパスワードに変更し、少なくとも8桁の長さに注意して、ファイルを保存してから、FlashAirをポップアップします。
